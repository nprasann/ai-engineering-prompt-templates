# Example: Legacy Modernization Project

Use `prompts/refactoring/safe-refactor-plan.md` with:

- `<PROJECT_NAME>`: `example-legacy-modernization`
- `<TECH_STACK>`: `Existing application stack`
- `<TASK>`: `Plan a behavior-preserving modernization of one module.`
- `<FILES_TO_INSPECT>`: `Target module, tests, build files, README`
- `<CONSTRAINTS>`: `No broad rewrites; preserve public behavior; add regression tests.`
- `<OUTPUT_FORMAT>`: `Risk-ranked refactor plan`
